# 🎓 Student Management System (Java CRUD)

A professional desktop application built with **Java Swing** and **MySQL** that allows users to manage student records efficiently. This project demonstrates a complete implementation of **CRUD operations** (Create, Read, Update, Delete) with a clean Graphical User Interface (GUI).

---

## 🚀 Features
- **Add Students:** Register new students with Name, Age, and Grade.
- **Update Records:** Modify existing student information easily.
- **Delete Entries:** Remove student records from the database.
- **Live Search:** Quick search functionality to find students by name.
- **Data Table:** Interactive table to display all student data in real-time.
- **Clear Fields:** Quick reset button to clear input fields.

---

## 🛠 Tech Stack
- **Frontend:** Java Swing (GUI)
- **Backend:** Java (JDK)
- **Database:** MySQL (via XAMPP)
- **Connector:** JDBC (MySQL Connector/J)

## 📂 Project Structure
- `src/`: Contains the Java source code.
- `database/`: Includes the `.sql` file to import the database schema.
- `lib/`: Contains the necessary JDBC drivers.

---

## ⚙️ How to Run
1. **Database Setup:**
   - Open XAMPP and start **Apache** and **MySQL**.
   - Go to `phpMyAdmin` and create a database named `student_db` (or your database name).
   - Import the SQL file located in the `/database` folder.

2. **Project Setup:**
   - Clone the repository: `git clone https://github.com/mohamedyassin1812/Java-MySQL-CRUD-System.git`
   - Open the project in **NetBeans** or your favorite IDE.
   - Add the `mysql-connector-java.jar` from the `lib` folder to your project's libraries.
   - Run `Main.java`.
